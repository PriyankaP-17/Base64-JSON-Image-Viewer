# ✨ Base64 JSON Image Viewer ✨

Transform your JSON files into a stunning visual gallery! This web app lets you **upload JSON files containing Base64-encoded images** and instantly preview them in a sleek, animated grid. Effortlessly **download individual images** or grab them all at once as a **ZIP file**.  

With a clean, modern interface and smooth interactions, managing Base64 images has never been this easy—or this fun!  

## Features
- Upload JSON files with Base64 images
- Responsive gallery view with animations
- Download individual images or all images as a ZIP
- Modern, user-friendly UI

## Built With
- HTML, CSS, JavaScript
- [JSZip](https://stuk.github.io/jszip/) for ZIP creation
- [FileSaver.js](https://github.com/eligrey/FileSaver.js/) for downloads
